O que são Single Page Applications

Single Page Applications são aplicações cuja funcionalidade está concentrada em uma única página. Ao invés de recarregar toda a página ou redirecionar o usuário para uma nova página,apenas o conteúdo principal é atualizado de forma assíncrona,assim oconteudo estetico da pagina é mantido

O que é uma API

API é um conjunto de padrões, rotinas e instruções de programação que permite que softwares ou aplicativos diferentes se conectem. Por meio de uma API é possível, fazer com que dois computadores “entendam” as instruções um do outro e gerem novas instruções a serem realizadas.
